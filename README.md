# 📥 Alur Pengumpulan Tugas

Mahasiswa wajib mengikuti alur berikut untuk mengumpulkan tugas.

## 1️⃣ Fork Repository

Fork repository ini ke akun GitHub masing-masing.

```
Fork → GitHub Anda
```

---

## 2️⃣ Buat Branch Berdasarkan NIM

Setelah fork, buat branch dengan nama **NIM**.

Contoh:

```
A2.2000104
```

---

## 3️⃣ Buat Folder Berdasarkan NIM

Di dalam repository hasil fork, buat folder:

```
A2.2000104
```

---

## 4️⃣ Buat Folder Praktikum

Di dalam folder NIM, buat folder untuk setiap praktikum.

Contoh:

```
Praktikum-01
Praktikum-02
Praktikum-03
```

---

## 5️⃣ Simpan Hasil Tugas

Semua file tugas praktikum disimpan di folder praktikum masing-masing.

Contoh file:

```
index.html
style.css
script.js
```

---

## 6️⃣ Push ke Repository Fork

Push hasil pekerjaan ke repository GitHub Anda.

```
git add .
git commit -m "Praktikum 01"
git push origin A2.2000104
```

---

## 7️⃣ Buat Pull Request

Buat **Pull Request** untuk merge ke repository utama.

```
branch NIM → master
```

---

## ✅ Status Tugas

Tugas dianggap **valid** apabila:

```
✔ Pull Request berhasil dibuat
✔ Struktur folder benar
✔ File tugas dapat dijalankan
✔ Pull Request di-merge oleh dosen
```

# 🔄 Sinkronisasi Fork

Tambahkan repository utama sebagai upstream:

```
git remote add upstream https://github.com/PBWFEND/SI-IV-B.git
```

Update repository fork:

```
git fetch upstream
git merge upstream/master
```
Panduan lengkap dapat dibaca di:

* https://help.github.com/en/articles/configuring-a-remote-for-a-fork
* https://help.github.com/en/articles/syncing-a-fork
