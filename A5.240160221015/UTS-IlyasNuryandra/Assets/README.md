## ✨ Features

| Feature | Detail |
|---|---|
| 🌙 Dark / Light Mode | Toggle dengan simpan preferensi di `localStorage` |
| 🎞️ Loading Animation | Animasi bar loading saat page pertama dibuka |
| 📱 Fully Responsive | Mobile, Tablet, Desktop — semua ukuran device |
| 🔍 Portfolio Filter | Filter proyek berdasarkan kategori (JS) |
| 🖼️ Lightbox Preview | Preview detail proyek tanpa reload halaman |
| 📋 Form Validation | Validasi real-time dengan error messages |
| 🚀 Lazy Loading | Native + polyfill untuk browser lama |
| ✨ CSS Animations | Reveal on scroll, orb, progress bars, cursor |
| 🖱️ Custom Cursor | Cursor khusus desktop dengan follower smooth |
| 🔗 Smooth Scrolling | Navigasi antar section tanpa snap keras |

## 🎨 Design System

| Token | Value |
|---|---|
| Font Display | Cormorant Garamond (serif) |
| Font Body | DM Sans |
| Font Mono | DM Mono |
| Primary Color | `#c9a96e` (Gold) |
| Background Dark | `#0a0a0a` |
| Background Light | `#f9f6f1` |

## 🛠️ Tech Stack

- **HTML5** — Semantic markup, accessible attributes
- **CSS3** — Custom Properties, Grid, Flexbox, Animations
- **JavaScript ES6+** — Vanilla JS, IntersectionObserver, localStorage
- **Font Awesome 6** — via CDN
- **Google Fonts** — Cormorant Garamond, DM Sans, DM Mono
